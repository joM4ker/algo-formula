# 二分

> 以下全部以使用闭区间为例

1. 明确二分序列。可能是数学上不单调的，但一定是一侧符合条件，另一侧不满足条件

2. 明确序列上，左侧还是右侧 是目标范围，基于左侧范围内的判断，写出check函数

3. 如果check为True，则l = mid + 1, 否则r = mid - 1

4. 结果为`l = r + 1 = 右侧范围的第一个值`

## 求最小/求最大

```pseudocode
def func(arr):
    def check(mid):
        mid是否满足左侧序列满足的条件
        
    l, r = 题意最小值, 题意最大值
    while l <= r:
        mid = l + (r - l) // 2
        if check(mid):
            l = mid + 1
        else:
            r = mid - 1
    return l
```
